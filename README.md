# Aubrisa Andes

**Andes** is a comprehensive data and reporting platform designed to complement and enhance 
core trading systems like Murex. It provides a collection of features to address common 
challenges in data accessibility, data integrity, reporting, and operational control.

This repository contains Infrastructure as Code (IaC) deployment templates.

These templates are intended as a guide and may be customized for specific security, 
resiliance and scale requirements.

## AWS

AWS deployment uses [CloudFormation](https://aws.amazon.com/cloudformation/) templates.

See [AWS Instructions](/aws/README.md)

## Azure

Azure deployment uses [Bicep](https://learn.microsoft.com/en-us/azure/azure-resource-manager/bicep/overview) templates.

See [Azure Instructions](/azure/README.md)
